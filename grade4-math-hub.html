Strategies and Algorithms

Area Model (Partial Products)

Break factors into place value parts and multiply each part:

23 √ó 47 = (20 + 3)(40 + 7) = 800 + 140 + 120 + 21 = 1,081

Builds understanding before standard algorithm.

Standard Algorithm

Fourth grade introduces the standard algorithm after conceptual foundations.

Key: Students should understand WHY it works, not just HOW.

üî¢ Fraction Equivalence

Building Understanding

The Big Idea

Equivalent fractions name the same amount using different-sized pieces.

1/2 = 2/4 = 4/8 - same amount, different number of pieces!

Generating Equivalents

Multiply numerator AND denominator by the same number (not zero).

This works because you're multiplying by 1 (e.g., 2/2 = 1).

Example: 3/4 = 3√ó2/4√ó2 = 6/8

Comparing Fractions

Strategy 1: Common denominators - convert to equivalent fractions

Strategy 2: Common numerators - compare piece sizes

Strategy 3: Benchmark fractions (compare to 1/2 or 1)

üìê Angles

Understanding Angle Measurement

What is an Angle?

An angle is formed by two rays sharing a common endpoint (vertex).

We measure how far one ray has rotated from the other.

Angle Types
  • Acute: Less than 90¬∞
  • Right: Exactly 90¬∞
  • Obtuse: Between 90¬∞ and 180¬∞
  • Straight: Exactly 180¬∞
Using Protractors

Key skills: Align vertex at center, one ray along 0¬∞, read correct scale.

Common error: Reading wrong scale (inside vs. outside numbers).

üîü Decimals

Introducing Decimal Notation

Connection to Fractions

Decimals are another way to write fractions with denominators of 10, 100, etc.

0.3 = 3/10 (three tenths)

0.47 = 47/100 (forty-seven hundredths)

Place Value Extension

Decimal point separates ones from tenths.

Each place is 10√ó less than the one to its left.

üìö Lesson 1: Multi-Digit Multiplication (Area Model)

‚è±Ô∏è 55 minutes

Learning Objectives

  • Students will multiply 2-digit by 2-digit numbers using the area model
  • Students will explain partial products and why they work

Lesson Procedure

Introduction (12 min): Review multiplication as area. "A garden is 23 ft by 14 ft. Let's find the area." Draw rectangle, break into parts: 20√ó10, 20√ó4, 3√ó10, 3√ó4.

Guided Practice (18 min): Calculate each partial product. Add together: 200 + 80 + 30 + 12 = 322 sq ft. Practice with another example.

Independent Practice (20 min): Solve problems using area model. Record all partial products.

Closure (5 min): "Why do we break numbers apart? How does this connect to place value?"

üìö Lesson 2: Equivalent Fractions

‚è±Ô∏è 50 minutes

Learning Objectives

  • Students will identify and generate equivalent fractions
  • Students will explain why equivalent fractions represent the same amount

Lesson Procedure

Introduction (10 min): Show 1/2 and 2/4 using fraction strips. "Are these the same amount?" Discuss: same size, different number of pieces.

Guided Practice (15 min): Discover pattern: 1/2 = 2/4 = 3/6 = 4/8. "What's happening to numerator and denominator?" Both multiplied by same number!

Independent Practice (20 min): Generate equivalent fractions. Find the missing number: 3/4 = ?/8.

Closure (5 min): "How many fractions equal 1/2? How do you know?"

üìö Lesson 3: Measuring Angles

‚è±Ô∏è 50 minutes

Learning Objectives

  • Students will use a protractor to measure angles
  • Students will classify angles as acute, right, obtuse, or straight

Lesson Procedure

Introduction (12 min): Review angle vocabulary. Introduce protractor parts: center point, baseline, degree marks. Model measurement.

Guided Practice (15 min): Practice measuring together. Key: vertex at center, ray along 0¬∞, read correct scale. Estimate first!

Independent Practice (18 min): Measure various angles, classify each, record measurements.

Closure (5 min): "How do you know which scale to read?"

üìö Lesson 4: Decimals and Fractions

‚è±Ô∏è 50 minutes

Learning Objectives

  • Students will convert between fractions with denominators 10/100 and decimals
  • Students will understand decimal place value (tenths, hundredths)

Lesson Procedure

Introduction (10 min): Show 10√ó10 grid. Shade 3 columns. "What fraction is shaded? 3/10. We write this as 0.3" Connect to place value.

Guided Practice (15 min): Practice tenths, then hundredths. 0.47 = 47/100. Locate on number line.

Independent Practice (20 min): Convert between forms. Shade grids for decimals. Write decimals from shaded grids.

Closure (5 min): "Is 0.8 or 0.75 larger? How do you know?"

üìö Lesson 5: Adding Fractions with Like Denominators

‚è±Ô∏è 45 minutes

Learning Objectives

  • Students will add fractions with the same denominator
  • Students will understand why only numerators are added

Lesson Procedure

Introduction (10 min): "I ate 2/8 of a pizza. My sister ate 3/8. How much did we eat together?" Use fraction strips.

Guided Practice (15 min): Model: 2/8 + 3/8 = 5/8. "We're adding eighths to eighths, so we still have eighths!" Add numerators, keep denominator.

Independent Practice (15 min): Practice adding. Include sums greater than 1: 5/6 + 4/6 = 9/6 = 1 3/6 = 1 1/2.

Closure (5 min): "Why don't we add the denominators too?"

üî¨ Investigation 1: Factor Game

Focus: Factors, multiples, prime/composite | Duration: 45 minutes

Procedure

  1. Use a 1-30 number grid
  2. Player 1 chooses a number (their score)
  3. Player 2 gets all unused factors of that number
  4. Alternate, choosing from remaining numbers
  5. After exploring, discuss: Which numbers give the most to your opponent? The least?
  6. Introduce prime (only 2 factors) vs. composite (more than 2 factors)

üî¨ Investigation 2: Fraction Equivalence Patterns

Focus: Generating equivalent fractions | Duration: 40 minutes

Procedure

  1. Create a "fraction family" for 1/2: list all equivalents up to twelfths
  2. Look for patterns: What's happening to numerators? Denominators?
  3. Predict: What would the equivalent with denominator 100 be?
  4. Repeat for 1/3 and 3/4
  5. Discussion: How can you tell if two fractions are equivalent?

Procedure

  2. Photograph or sketch each angle found
  3. Estimate measurement, then measure with protractor
  4. Classify each: acute, right, obtuse, straight
  5. Challenge: Find an angle of exactly 45¬∞, 90¬∞, 120¬∞
  6. Create a class angle gallery

üî¨ Investigation 4: Decimal Number Line

Focus: Decimal place value | Duration: 40 minutes

Procedure

  1. Create a large number line from 0 to 1
  2. First, mark tenths (0.1, 0.2, ... 0.9)
  3. Between 0.3 and 0.4, mark hundredths
  4. Place mystery decimals: 0.35, 0.07, 0.72

üìö What Are Equivalent Fractions?

Imagine cutting a pizza. If you cut it into 2 equal pieces and eat 1, you ate 1/2. But if you cut it into 4 pieces and eat 2, you still ate the same amount - just in smaller pieces!

1/2 = 2/4. These are called equivalent fractions. They name the same amount using different-sized pieces.

To make equivalent fractions, multiply (or divide) the top and bottom by the same number. Try it: 3/4 = 6/8. Same pizza, different slices!

üìö Understanding Angles

An angle measures how much something has turned. Think of a door opening - it can open a little (small angle) or a lot (big angle).

We measure angles in degrees. A complete turn is 360¬∞. A right angle (like a corner of a book) is 90¬∞. Half a turn is 180¬∞.

Angles smaller than 90¬∞ are called acute. Angles larger than 90¬∞ but smaller than 180¬∞ are obtuse. Look around - angles are everywhere!

üìö Decimals: Another Way to Write Fractions

You see decimals every day! Money uses decimals: $3.50 means 3 dollars and 50 cents (50 hundredths of a dollar).

The decimal point separates wholes from parts. After the point, the first place is tenths, the second is hundredths.

0.7 = 7/10 (seven tenths). 0.25 = 25/100 (twenty-five hundredths). Same amounts, different ways to write them!

üìñ Math Vocabulary

Equivalent Fractions

Definition: Fractions that represent the same amount

"1/2, 2/4, and 3/6 are all equivalent fractions."

Prime Number

Definition: A number with exactly 2 factors: 1 and itself

"7 is prime because only 1 √ó 7 = 7."

Composite Number

Definition: A number with more than 2 factors

"12 is composite: 1, 2, 3, 4, 6, 12 are all factors."

Acute Angle

Definition: An angle measuring less than 90¬∞

"A 45¬∞ angle is acute."

Obtuse Angle

Definition: An angle measuring more than 90¬∞ but less than 180¬∞

"A 120¬∞ angle is obtuse."

Decimal

Definition: A number written with a decimal point showing parts less than 1

"0.75 is a decimal equal to 75/100 or 3/4."

Tenths

Definition: The first place to the right of the decimal point

"In 0.3, the 3 is in the tenths place."

Hundredths

Definition: The second place to the right of the decimal point

"In 0.47, the 7 is in the hundredths place."

‚ùì Common Misconceptions

‚ùå Multiply numerator AND denominator to add fractions

Error: 1/4 + 2/4 = 3/8

Correction: Same denominator = same-sized pieces. Just add pieces: 1/4 + 2/4 = 3/4

‚ùå 0.45 > 0.6 because 45 > 6

Error: Comparing digits without considering place value

Correction: 0.6 = 0.60 = 60 hundredths. 60 > 45, so 0.6 > 0.45

‚ùå Reading wrong protractor scale

Error: Reading 150¬∞ instead of 30¬∞ (or vice versa)

Correction: Always estimate first. Does 150¬∞ make sense for an acute angle? Check angle type.
